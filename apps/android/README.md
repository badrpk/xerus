# Android app

The official Android client for this product lives in the monorepo:

**https://github.com/badrpk/android-apps**

Module: see `apps/xerus/` in that repository.

## Features
- Sign up / Sign in
- OTP for first-time users only
- Existing verified users **skip OTP**
- Google & Facebook login without OTP
- GCP API key via `local.properties` (never commit)

## Build
Open the monorepo in Android Studio, select this app module, Run.
